### 示例 1：普通描述 vs 文学技巧润色

**原文（无润色）**：
> NotebookLM 是 Google 推出的知识管理工具。它可以帮助用户整理文档，回答问题。

**润色后（金庸式笔触 + 列锦手法）**：
> 想象一下，在江湖中，若有一位绝世高手，能够过目不忘，将天下武学秘籍尽收脑海，并能举一反三，发现不同武学之间的关联。这便是 Google 推出的 NotebookLM，一位当世的知识管理"大宗师"。

它深谙"书山有路勤为径"的道理，能助你：
- 📚 整理文档，如同管理藏经阁
- 🤔 回答问题，如同指点武功
- 🔗 发现关联，如同融会贯通

### 示例 2：普通描述 vs 数字化表达

**原文（无润色）**：
> AI 在2025年有很多应用场景，比如客户服务、内容创作、数据分析等。

**润色后（数字化表达 + 列锦手法）**：
> 站在2025年的门槛上，AI正以"一日千里"的速度演进。让我们看看它带来的三大变革：

**数据**显示：80%的客户咨询已由AI处理，内容创作效率提升**300%**，数据分析准确率达**95%**以上。

## 🚀 实际使用命令

### 完整工作流程（推荐）

```bash
# 步骤1: 使用 polishing-content 润色文章（核心！）
# 在 Claude Code 中执行：
请润色 output/articles/01-ai-skills-notebooklm.md

# 步骤2: 生成图片
python3 skills/wechat-workflow/generating-article-images/scripts/generate_images.py \
    --input "output/articles/01-ai-skills-notebooklm.md"

# 步骤3: 转换为 HTML
python3 skills/wechat-workflow/converting-to-wechat/scripts/convert.py \
    --input "output/articles/" \
    --output "output/html/"
```

### 批量处理多篇文章

```bash
# 步骤1: 批量润色
请润色 output/articles/ 目录下所有的 md 文件

# 步骤2: 批量生成图片
for file in output/articles/*.md; do
    python3 skills/wechat-workflow/generating-article-images/scripts/generate_images.py --input "$file"
done

# 步骤3: 批量转换
python3 skills/wechat-workflow/converting-to-wechat/scripts/convert.py \
    --input "output/articles/" \
    --output "output/html/"
```

### 高级用法

```bash
# 润色并验证技术准确性
请润色 output/articles/01-ai-skills-notebooklm.md，并验证所有代码片段的技术准确性

# 润色指定技术领域的文章
```

## ⚠️ 常见错误

### 错误 1：跳过 polishing-content

```bash
# ❌ 错误做法：直接生成图片
python3 generate_images.py --input article.md

# ✅ 正确做法：先润色，再生成图片
请润色 article.md
python3 generate_images.py --input article.md
```

### 错误 2：手动添加图片提示词

```bash
# ❌ 错误做法：手动写图片提示词
# 容易遗漏、格式不一致、质量不稳定

# ✅ 正确做法：使用 polishing-content 自动生成
请润色 article.md，并在适当位置添加图片生成提示词
```

### 错误 3：只润色不生成图片

```bash
# ❌ 错误做法：润色后直接转换 HTML
# 文章没有图片，视觉效果差

# ✅ 正确做法：润色 → 生成图片 → 转换 HTML
请润色 article.md
python3 generate_images.py --input article.md
python3 convert.py --input article.md
```
